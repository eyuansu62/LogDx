## Possible Root Cause
The build failed because `wget` returned `HTTP/1.1 502 Bad Gateway` when attempting to download the nydus release asset from GitHub, which caused the Docker layer build to exit with code 1 and cancel subsequent steps. This points to a transient or persistent network/remote-host issue (release asset unavailable or proxy/GitHub outage) rather than an internal build script error.
